## 🛠 How Freelancer Works: Step-by-Step Guide

### For Freelancers

1. **Create a Free Profile**  
   Add your skills, experience, and portfolio.

2. **Browse Jobs**  
   Filter by skills, price, or time left to bid.

3. **Submit a Proposal**  
   Write a pitch, set your price, and include any relevant questions.

4. **Communicate via Chat**  
   Use real-time messaging and file sharing for updates.

5. **Work and Deliver**  
   Track milestones and deliver work through the platform.

6. **Get Paid**  
   Withdraw via PayPal, bank transfer, or Payoneer.

---

### For Employers

1. **Post a Project**  
   Describe your job, select skills, and set a budget.

2. **Review Proposals**  
   Choose from vetted freelancers.

3. **Award the Project**  
   Set milestones and start collaboration.

4. **Manage and Pay**  
   Use the dashboard to release milestone payments as you go.

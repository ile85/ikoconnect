## 🎨 What is Figma?

**Figma** is a cloud-based **interface design tool** used for **UI/UX design**, prototyping, wireframing, and collaboration. It runs entirely in the browser and supports real-time multiplayer editing, making it ideal for teams, freelancers, and developers.

Unlike tools like Adobe XD or Sketch, Figma doesn’t require downloads or licenses—just a link and a browser.

---

## ✨ Why Designers Love Figma

- 👥 **Real-time Collaboration** – co-edit designs just like Google Docs
- 🔁 **Version History** – see every change ever made, by anyone
- 🎯 **Design Systems** – create reusable components and style libraries
- 💻 **Cross-platform** – works on Mac, Windows, Linux, or Chromebook
- 📱 **Prototyping & Animation** – build interactive mockups natively
- 🔗 **Developer Handoff** – generate code snippets automatically
- 🔒 **Commenting & Feedback** – right inside the canvas

## 🚀 Quick Start Guide

### 1. Sign Up Free  
Go to [figma.com](https://www.figma.com/) and create a free account.

### 2. Start a New File  
You can choose **Design**, **Whiteboard**, or import an existing `.fig` file.

### 3. Use the Toolbar  
Access tools like Frame, Text, Shapes, Auto Layout, Components.

### 4. Invite Team Members  
Click “Share” and add collaborators by email or public link.

### 5. Export Assets  
Select objects → Right-click → Export → PNG, SVG, or PDF.

---

## 📐 Key Features Explained

### 🔁 Components & Variants  
Reusable UI pieces like buttons, cards, modals—with different states (hover, active).

### 📏 Auto Layout  
Auto-adjusts spacing between elements dynamically—super useful for responsiveness.

### 🌐 Design Systems  
Shared styles (fonts, colors, effects) that can be applied across files.

### 🔍 Inspect Mode  
Click any element and get CSS, iOS, or Android code automatically.

### 🔗 Plugin Ecosystem  
Tons of plugins like:
- Unsplash (insert free images)
- Iconify (100k+ icons)
- Content Reel (fake content)
- Contrast Checker (accessibility)

## 💡 Pro Tips

- Use `Cmd/Ctrl + /` to search for any command
- Use `Shift + A` to quickly wrap elements in Auto Layout
- Use `Shift + R` to toggle rulers
- Turn on "Snap to pixel grid" for sharp designs
